# Alila Engineering - Corporate Website 

A modern, fully responsive, and bilingual corporate web application developed for Alila Engineering. This project aims to reflect the company's innovative engineering solutions and sustainable design vision in the digital world.

## Live Demo
You can view the live version of the website here: **[https://alila-muhendislik.netlify.app/]**

## Key Features
* **Bilingual Support (TR/EN):** Fully translated infrastructure including menus, content, and dynamic components.
* **Dynamic Portfolio (JSON):** The projects section is populated dynamically using a `projects.json` file, allowing easy updates without touching the HTML structure.
* **Responsive Design:** Optimized for all devices (Mobile, Tablet, Desktop) using modern CSS Flexbox and Grid layouts.
* **Interactive Filtering:** Users can filter engineering projects by category (Infrastructure, Superstructure, Restoration, etc.) instantly.
* **Working Contact Form:** Integrated with Formspree API to receive direct emails without needing a backend server.

## Technologies Used
* **Frontend:** HTML5, CSS3, Vanilla JavaScript (ES6+)
* **Data Management:** JSON
* **Icons & Typography:** FontAwesome, Google Fonts
* **Deployment:** Netlify
* **Form Handling:** Formspree API

## Development Process
This project was built from scratch without relying on heavy frameworks (like Bootstrap or React) to ensure maximum performance, clean code architecture, and fast load times. The dynamic data fetching logic ensures that the company can easily add new projects in the future just by updating a single JSON file.
